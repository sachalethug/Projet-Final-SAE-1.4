Comment rejoindre le serveur ?

Pour cela vous pouvez rejoindre via plusieurs moyens :

Sur navigateur :

1. Rendez-vous sur le site eaglercraft.com/mc/1.8.8-wasm/
2. Créer votre profil

Sur launcher :

1. Lancer votre client Minecraft
2. Connectez vous / rentrez votre nom d'utilisateur

Maintenant, les prochaines étapes sont communes pour les deux types de clients :

3. Cliquer sur "Multiplayer"
4. Cliquer sur "New Server"
5. Ajouter l'adresse suivante selon votre accès :


Navigateur (Eaglercraft) : wss://eagler.rtcraft.fr

Client Minecraft : rtcraft.fr

Vous êtes maintenant connecté sur le serveur, vous devez exécuter la commande /register pour pouvoir enregistrer votre nom d'utilisateur avec la syntaxe suivante : /register "MOTDEPASSE" "MOTDEPASSE"

La prochaine fois que vous vous connectez vous devrez utiliser la commande /login suivie de votre mot de passe :

/login "MOTDEPASSE"

Bravo ! Vous êtes maintenant prêt à jouer sur le serveur !
